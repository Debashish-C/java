## Directory Structure

- `Lab1_BasicConcepts/` - Basic Java concepts, data types, loops, arrays
- `Lab2_ClassesAndObjects/` - Classes, objects, constructors
- `Lab3_Inheritance/` - Inheritance and its types
- `Lab4_Polymorphism/` - Method overloading and overriding
- `Lab5_AdvancedConcepts/` - Interfaces, packages, multithreading, applets, JDBC

## How to Compile and Run

1. Make sure you have JDK installed
2. Navigate to the specific lab directory
3. Compile: `javac FileName.java`
4. Run: `java FileName`

For JDBC programs:
- MySQL JDBC driver is required
- Configure database connection details in the code

For Applets:
- Use appletviewer or a browser that supports Java applets
- HTML files are provided for applet demonstration
